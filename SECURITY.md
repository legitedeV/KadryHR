# Security Policy

## Reporting a Vulnerability

Please report suspected vulnerabilities privately via: **contact@kadryhr.pl**.

When reporting, include:
- concise impact summary,
- reproduction steps,
- affected scope (service/module).

Do **not** include PII or sensitive production data in reports.

## Disclosure Policy

- We acknowledge new reports within **72 hours**.
- We triage severity and assign remediation targets.
- We coordinate disclosure timing with the reporter where possible.

## Scope

Security reporting scope includes:
- `https://kadryhr.pl`
- API endpoints served under `https://kadryhr.pl/api/...`

## Public Issues

Please do not post vulnerabilities or PII in public GitHub issues.
Use the private reporting channel above.
